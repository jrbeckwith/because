because.hosts module¶

Define locations and metadata for hosts running public BCS services.

This module contains only a dictionary called HOSTS. Each key is a short alias for a host giving its role, e.g. “dev”, “test”, or “prod”. Each value is an instance of boundless.service.Host, containing a base URL and any associated information necessary to use it, aside from information about services and endpoints.

because.hosts.HOSTS = {'local': <because.service.Host object>, 'dev': <because.service.Host object>, 'test': <because.service.Host object>, 'prod': <because.service.Host object>}¶
