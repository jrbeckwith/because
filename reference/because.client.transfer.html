because.client.transfer module¶

class because.client.transfer.Transfer(request, ssl_config=None, log=None)[source]¶

Bases: object

Hold per-request state for one ongoing HTTP request cycle.

This can play a role like a future or promise in implementations that support asynchronous network I/O.

This can also be used as a device for bundling up multiple descriptions of a transfer’s state to be handled by e.g. one callback, instead of requiring many separate callbacks to handle many different state notifications.

close()[source]¶

Abort the transfer, if possible.

If aborting is not possible, then flail your arms like a windmill and announce your intent to ignore any response becuase it’s too late now, the crucial moment has already passed and you just can’t anymore.

log = <logging.Logger object>¶

Default logger to use if no logger was passed.

start()[source]¶

Begin the transfer.

This method allows execution of a transfer to be deferred after its construction, which couldn’t be done if __init__ started it.
