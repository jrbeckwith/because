class because.client.response.Response(status, headers=None, body=b'')[source]¶

Bases: because.base.utils.PrettyPrintable

Represent a received HTTP response.

You may be saying to yourself: “why yet another class Response?!?” Well – this internal representation is used to abstract over representations like httplib.HTTPResponse or QNetworkReply, so that because can work about the same in multiple environments without all the response processing code having to be duplicated with minor variations in it.
